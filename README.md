# Gold Scalper Bot - MQL5 Expert Advisor

An aggressive 5M/15M breakout scalper for XAUUSD with Fibonacci retracements.

## Features
- **3 Trading Modes**: Scalp (2-5 pips), Swing (10-20 pips), Trend Follow
- **Control Panel**: Mode switching, lot size adjustment, risk management
- **Fib Breakout Detection**: Swing highs/lows + Fib retracements
- **Risk Management**: Position sizing, stop losses, take profits
- **XM Global MT5 Compatible**: Ready for demo/live trading

## Installation
1. Clone this repo
2. Copy `GoldScalperBot.mq5` to `MQL5/Experts/` folder in MT5
3. Restart MT5
4. Attach EA to XAUUSD chart (5M timeframe recommended)
5. Adjust settings in EA inputs

## Usage
- Select trading mode in EA inputs
- Adjust lot size or risk % per trade
- Monitor control panel for signals
- Demo account recommended first
